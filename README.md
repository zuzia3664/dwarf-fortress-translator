## 📖 About this application

Dwarf Fortress contains deep systems and complex text. Players often struggle to parse this information when English is not their native language. This software provides real-time translation for the game interface. It uses optical character recognition to read screen text and translates it instantly into your preferred language through Google Translate. The tool displays the translated text in a transparent overlay. You read the information without leaving the game window.

## 🛠 Features

*   **Real-time scanning:** The software tracks the game screen and updates text instantly.
*   **Broad translation support:** Choose from over 20 languages.
*   **Non-intrusive design:** The overlay sits on top of the game window.
*   **Simple controls:** Toggle the display with a single hotkey.
*   **Low resource usage:** The program keeps your frame rates stable while you play.

## 🔍 Troubleshooting

**The text looks garbled**
Ensure the game window is in windowed mode or borderless windowed mode. Fullscreen mode can block the scan tool from seeing your game text. Check that the scan area box covers the text clearly.

**The translation does not update**
Ensure your internet connection remains active. The software requires a connection to reach the Google Translate service. Check your firewall settings to verify that the application has permission to access the web.

**The program slows down my game**
Close background applications like web browsers or video players. These programs compete for memory. You can also lower the scan frequency in the application settings to reduce the impact on your processor.

**The overlay disappears**
Press the activation key again to toggle the overlay. If it does not return, restart the application.
